O que é HTML

(inicio)

A linguagem de Marcação de hipertexto (HTML) é uma lingugem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem paras os servidores da web qual é o estilo e aestrutura de um documento.

Links

(inicio)

É uma tag vazia, que contém apenas atributos e faz a relação do documento HTML com recurso externos, é comumente usado para vincular uma folha de estilo externa, também é usada para definir o favicon da página(ícone da aba do navegador), como outros recursos.

Listas ordenadas e nao ordenadas
(inicio)

lista ordenadas

Uma lista ordenada começa com tag "ol" e seus respectivos itens da lista ficam dentro da tag "li". Por padrão as listas ordenadas por números, mas também iremos ver como ordena-la por outros metodos.

listas não ordenadas

já as listas desordenadas são iniciadas com a tag "ul" e são representadas por pequenos "bullets".
